10 Common Linux Commands

1.) ls

The "ls" command allows the terminal to list all the files that are located in the current directory that you're in.
Example: ls

2.) cd

The "cd" command allows you to switch to a different directory. That way you can work on a different directory such as a website directory, without influencing any other directory.
Example: cd website

3.) nano

This command allows you to open a file such as, if you're working on a website, opening a website releted file will give you access to the html scripts of the website.
Example: nano secondpage.html

4.) cp

This command allows you to copy a file and move it some other location if you want to.
Example: cp /etc/passwd student4

5.) mkdir

This command allows you to make a new directory.
Example: mkdir folder

6.) rmdir

This command allows you to remove any directory.
Example: rmdir folder

7.) mv

This command allows you to move a file to another location
Example: mv website

8.) echo

This command allows you to return or repeat a message
Example: echo HelloWorld

9.) rm

This command allows you to remove or permanently delete a file
Example: rm folder

10.) clear

This command allows you clear everything previously on the prompt page.
Example: Just type "clear" on the command line
